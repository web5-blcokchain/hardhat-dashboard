
# 前端界面模块

## 模块说明
基于 Vue3 的前端监控界面，提供节点状态和日志的可视化展示。

## 功能特性
- 节点状态实时监控
- 日志实时展示
- 响应式设计
- 主题定制

## 配置说明
- 开发服务器：5173端口
- API配置
- 主题配置

## 使用方法
```bash
yarn install
yarn dev
```

## 开发指南
- 组件在 components 目录
- 页面在 views 目录
- 样式在 styles 目录
