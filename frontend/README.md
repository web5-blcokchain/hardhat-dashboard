# 前端界面模块

[返回主项目](../README.md)

## 模块说明
基于 Vue3 的前端监控界面，提供节点状态和日志的可视化展示。

## 功能特性
- 节点状态实时监控
- 日志实时展示
- 响应式设计
- 主题定制

## 配置说明
- 开发服务器：5173端口
- API配置：config.js
- 主题配置：tailwind.config.js

## 使用方法
```bash
# 单独运行
yarn install
yarn dev

# 通过主项目启动脚本运行
cd ..
node start.js
```

## 开发指南
- 组件在 components 目录
- 页面在 views 目录
- 样式在 styles 目录
